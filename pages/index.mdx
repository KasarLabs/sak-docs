# Welcome to Starknet Agent Kit

Starknet Agent Kit is a powerful toolkit for creating AI agents that can interact with the Starknet blockchain. It combines the power of AI language models with blockchain capabilities, allowing you to build intelligent applications that can perform complex operations on Starknet.

## Example Usage

```typescript
import { StarknetAgent } from 'starknet-agent-kit';

// Initialize agent
const agent = new StarknetAgent({
  aiProviderApiKey: 'your-ai-provider-key',
  aiProvider: 'anthropic',
  aiModel: 'claude-3-5-sonnet-latest',
  accountPrivateKey: 'your-wallet-private-key',
  rpcUrl: 'your-rpc-url',
});

// Execute natural language commands
await agent.execute('Transfer 0.1 ETH to 0x123...');
await agent.execute('What is my USDC balance?');
```
