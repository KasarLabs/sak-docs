# Getting Started with Starknet Agent Kit

## Installation

Install the Starknet Agent Kit and its peer dependencies:

```bash
npm install starknet-agent-kit @nestjs/common @nestjs/core starknet @langchain/anthropic
```

## Basic Setup

```typescript
import { StarknetAgent } from 'starknet-agent-kit';
import { RpcProvider } from 'starknet';

// Initialize provider
const provider = new RpcProvider({
  nodeUrl: process.env.RPC_URL
});

// Create agent
const agent = new StarknetAgent({
  provider,
  accountPublicKey: process.env.PUBLIC_KEY,
  accountPrivateKey: process.env.PRIVATE_KEY,
  aiProvider: 'anthropic',
  aiModel: 'claude-3-5-sonnet-latest',
  aiProviderApiKey: process.env.AI_PROVIDER_API_KEY
});
```

## Environment Setup

Create a `.env` file with the following variables:

```env
# Blockchain Configuration
PRIVATE_KEY=your_private_key
PUBLIC_ADDRESS=your_public_address
RPC_URL=your_rpc_url

# AI Provider Configuration
AI_PROVIDER_API_KEY=your_ai_provider_key
AI_MODEL=claude-3-5-sonnet-latest
AI_PROVIDER=anthropic
```

## Basic Usage

Execute your first operations:

```typescript
// Check balance
const balance = await agent.execute('What is my ETH balance?');

// Send tokens
await agent.execute('Send 0.1 ETH to 0x123...');

// Perform a swap
await agent.execute('Swap 5 USDC for ETH');
```
