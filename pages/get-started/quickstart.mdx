# Quickstart Guide

This guide will help you get started with Starknet Agent Kit quickly. We'll cover installation, basic setup, and your first agent interaction.

## Prerequisites

Before you begin, make sure you have:

- Node.js (v16 or later)
- PNPM package manager
- A Starknet wallet and private key
- An AI provider API key (Anthropic, OpenAI, Google, or Ollama)

## Installation

1. Install the package via npm:

```bash
npm install starknet-agent-kit
# or with pnpm
pnpm add starknet-agent-kit
```

2. Install required peer dependencies:

```bash
npm install @nestjs/common @nestjs/core @nestjs/platform-fastify starknet @langchain/anthropic
```

## Basic Setup

1. Create an environment file (.env):

```env
PRIVATE_KEY="your-starknet-private-key"
PUBLIC_ADDRESS="your-starknet-address"
AI_PROVIDER_API_KEY="your-ai-provider-key"
AI_MODEL="claude-3-5-sonnet-latest"
AI_PROVIDER="anthropic"
RPC_URL="your-starknet-rpc-url"
```

2. Initialize your agent:

```typescript
import { StarknetAgent } from 'starknet-agent-kit';

const agent = new StarknetAgent({
  provider: new RpcProvider({ nodeUrl: process.env.RPC_URL }),
  accountPrivateKey: process.env.PRIVATE_KEY,
  accountPublicKey: process.env.PUBLIC_ADDRESS,
  aiModel: process.env.AI_MODEL,
  aiProvider: process.env.AI_PROVIDER as 'anthropic',
  aiProviderApiKey: process.env.AI_PROVIDER_API_KEY,
});
```

## Your First Agent Interaction

Here's a simple example of using your agent:

```typescript
// Basic balance check
const balanceResponse = await agent.execute("What's my ETH balance?");
console.log(balanceResponse);

// Token transfer
const transferResponse = await agent.execute(
  "Transfer 0.1 ETH to 0x1234...5678"
);
console.log(transferResponse);

// DeFi operation
const swapResponse = await agent.execute(
  "Swap 10 USDC for ETH on Avnu"
);
console.log(swapResponse);
```

## Using Individual Tools

You can also use individual tools directly:

```typescript
import { 
  getBalance, 
  transfer, 
  swapTokens 
} from 'starknet-agent-kit';

// Get balance directly
const balance = await getBalance({
  walletAddress: '0x123...',
  assetSymbol: 'ETH'
});

// Transfer tokens
const result = await transfer({
  recipient_address: '0x456...',
  amount: '0.1',
  symbol: 'ETH'
});
```

## Troubleshooting

Common issues and their solutions:

- **RPC Connection Issues**: Ensure your RPC URL is correct and accessible
- **Transaction Failures**: Check wallet balance and transaction parameters
- **AI Model Errors**: Verify your API key and model name are correct
